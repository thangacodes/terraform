# How to clone and run this IaC script:

1. Clone the repository:
   
   git clone https://github.com/thangacodes/terraform.git

2. Navigate to the EC2 module folder:

   cd terraform/ec2-module  

3. Initialize Terraform:

   terraform init

   terraform plan
