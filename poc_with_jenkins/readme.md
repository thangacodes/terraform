Terraform IaC for web application deployment using a bastion host as a proxy for Jenkins CI/CD.

Pre-requisites:

A minimum of three (3) dedicated Amazon EC2 instances with basic CPU, memory, and storage specifications suitable for the deployment.
Each EC2 instance must be assigned a public IP address.
Appropriate ingress rules must be configured and enabled in the respective security groups to allow required network access.
